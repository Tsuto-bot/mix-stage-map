# mix-stage-map

Mix Stage Mapは、ミックスの音の配置（左右・奥行き・広がり）を視覚的に整理するた
めのツールです。
1. 画面の構成
画面は大きく3つのエリアで構成されています。

左　　トラック管理（追加・削除・名前変更・色変更・Pan / Depth / サイズ調整）
中央　Stage（音の配置を表示するステージ）
下　　Sections（曲構成ごとの配置を保存）

2. トラック操作
• 追加：左上の「追加」ボタンをクリック
• 選択：左のトラック一覧をクリック
• 名前変更：「名前」欄を編集
• 色変更：「色」欄でカラーを変更

3. ステージ操作
• 位置変更：円をドラッグ
• サイズ変更：Shiftキーを押しながらドラッグ

4. セクション操作
• セクション切り替え：下のカードをクリック
• セクション追加：「セクション追加」をクリック
• セクションコピー：コピー元 → コピー先 を選択して「コピー」

5. Todo機能
セクションごとにミックス作業を整理するためのメモ機能です。各セクションにTodoを
追加して、作業内容を管理できます。
■ Todoを追加する
セクションカード下部の「Add
todo」入力欄に作業内容を入力し、Enterキーまたは「＋」ボタンを押すとTodoが追加されます。
■ Todoを完了する
Todoの左側にあるチェックボックスをクリックすると完了状態になります。完了すると文字がグ
レーになり、取り消し線が表示されます。
■ Todoを編集する
Todoのテキストをダブルクリックすると編集できます。Enterキーで保存、Escキーで編集をキャン
セルできます。
■ Todoを削除する
編集状態で内容を空にしてEnterキーを押すと、そのTodoは削除されます。

6. 保存 / 読み込み
• 保存：「保存」ボタン → JSONファイルとして保存
• 読込：「読込」ボタン → 保存したJSONを選択

7. トラック並び替え
左のトラック一覧をドラッグすると順番を変更できます。

8. トラックのデフォルト機能
よく使うトラックの「名前」と「色」を、このブラウザのデフォルトとして保存できま
す。
• ① トラックの名前と色を設定
• ② 「名前と色をデフォルト保存」をクリック
• ③ 次回アプリ起動時にその設定が自動で読み込まれます
※ 保存されるのは「名前」と「色」のみです。Pan・Depth・位置などは保存されません。
